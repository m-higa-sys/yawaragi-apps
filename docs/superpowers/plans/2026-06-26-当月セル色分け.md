# ①②③ 当月セル色分け Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** ①個訓・②口腔・③通所の各表で「今月どの提出セルをやるか」を統一3色（赤=未作成/緑=作成済未送付/青=送付済）で可視化し、①②③を shared.js §I 参照に切替える。

**Architecture:** フロントのみ（GAS無書込・既存GET読み取り）。色判定は §I の純関数 `submitCellColor` に一本化しTDD。①②③は当月セルでそれを呼ぶ。当月セルのみ着色・他月は従来（無回帰）。

**Tech Stack:** 素のHTML/JS、node（純関数TDD）。

**設計書:** `docs/superpowers/specs/2026-06-26-当月セル色分け-design.md`

---

## 統一CSS（①②③に同一値で定義する）

```css
/* 当月の提出セル 3色（赤=未作成/緑=作成済未送付/青=送付済）。当月セルのみに付与 */
.cell-todo  { background:#ffebee !important; box-shadow: inset 0 0 0 2px #e53935 !important; }
.cell-ready { background:#e8f5e9 !important; box-shadow: inset 0 0 0 2px #43a047 !important; }
.cell-done  { background:#e3f2fd !important; box-shadow: inset 0 0 0 2px #1e88e5 !important; }
```

クラス対応：`submitCellColor` の戻り値 `red→cell-todo` / `green→cell-ready` / `blue→cell-done` / `''→付与しない`。

// 当月提出セルの色: 非該当→''／送付済→'blue'／作成済未送付→'green'／該当未作成→'red'。
// ①②③の当月セル色分けで共通利用（状態→色の一本化）。
function submitCellColor(isApplicable, hasCreated, hasSent) {

}
