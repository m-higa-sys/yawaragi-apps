# 士業提出物 保管ルール台帳 v1

税理士（アシタエ・篠崎彰人先生）／社労士（タスクマン・朝野さん）への月次・四半期提出物を、
**どこから取り／どこへ出し／どこへ保管するか**を1枚にまとめた台帳。
朝の報告（morningDigest）の定例タスクと対応。

- **保管先の正**：Google Drive に集約
- **控えの方針**：ChatWork で送ったものも、自分用の控えを必ず Drive に保管する
- **命名規則**：`年月_内容.拡張子`（例 `2026-07_勤務実績CSV.csv`）／四半期は `年Q四半期`（例 `2026Q3`）
- **年フォルダで区切る**：7年保存を見据え、翌年は `2027/` に同じ規則で積む
- 最終更新：2026-07-09

## フォルダ構成（Google Drive）

```
士業提出物/
├── 社労士_朝野さん/
│   └── 2026/
│       ├── 2026-07_勤務実績CSV.csv
│       └── 2026-07_処遇改善加算お知らせ.pdf
├── 税理士_アシタエ/
│   └── 2026/
│       ├── 2026-07_アシタエ12ファイル/      ← フォルダごと控え
│       └── 2026Q3_領収書3区分/             ← 四半期・スキャン控え
└── 経理_月次書類/
    └── 2026/
        └── 2026-07_支払決定額通知書.pdf
```

考え方は3つだけ：**提出先で分ける**／**年で区切る**／**ファイル名は `年月_内容`**。

## タスク別 保管ルール（5件）

### 1. 勤務実績CSV送付（kinmu-csv）
- **提出先**：社労士 朝野さん（タスクマン）／ChatWork
- **取得元**：ケアズ CSVエクスポート
- **締切**：毎月10日（開始1日）
- **保管先**：`士業提出物/社労士_朝野さん/2026/`
- **控えファイル名**：`2026-07_勤務実績CSV.csv`
- **手順**：ケアズからCSVをDL → ChatWorkで朝野さんへ送付 → **同じファイルを上記フォルダに控え保管**

### 2. 処遇改善加算お知らせDL→転送（carezou-shoguu）
- **提出先**：社労士 朝野さん
- **取得元**：けあ蔵（配信21〜23日頃）
- **締切**：毎月25日（開始20日）
- **保管先**：`士業提出物/社労士_朝野さん/2026/`
- **控えファイル名**：`2026-07_処遇改善加算お知らせ.pdf`
- **手順**：けあ蔵からお知らせをDL → 朝野さんへ転送 → 上記フォルダに控え保管

### 3. アシタエ12ファイル送付（ashitae-package）
- **提出先**：税理士 篠崎彰人先生ルーム（ChatWork）
- **取得元**：Drive 月別フォルダ（対応表＝請求フロー_月次チェックリスト_v4.md）
- **締切**：毎月25日（開始20日）
- **保管先**：`士業提出物/税理士_アシタエ/2026/2026-07_アシタエ12ファイル/`（**フォルダごと**）
- **手順**：12ファイルを揃える → ChatWorkで篠崎ルームへ送付 → **送った12ファイル一式を上記フォルダにコピー保管**
- ※単票ではなくフォルダ単位。中身は請求フロー_月次チェックリスト_v4.md の対応表に従う

### 4. 支払決定額通知書・内訳書DL（carezou-tsuchisho）
- **提出先**：（自社経理保管。士業へ直接出すものではない）
- **取得元**：けあ蔵 国保伝送メニュー→通知文書（配信20〜23日）
- **締切**：毎月25日（開始20日）
- **保管先**：`士業提出物/経理_月次書類/2026/`
- **控えファイル名**：`2026-07_支払決定額通知書.pdf`
- **手順**：けあ蔵から通知書・内訳書をDL → 上記フォルダに保管

### 5. 領収書3区分の原本提出（ryoshusho-3kubun）
- **提出先**：税理士（アシタエへ郵送/手渡し）
- **取得元**：領収書原本（現金／クレカ／通帳の3区分・3ヶ月分）
- **締切**：四半期の25日（1・4・7・10月）※開始1日
- **保管先**：`士業提出物/税理士_アシタエ/2026/2026Q3_領収書3区分/`
- **手順**：原本は郵送/手渡しで提出 → **スキャン/撮影した控えを上記フォルダに保管**
- ※原本は税理士へ、手元にはスキャン控えのみ残る

## 補足メモ

- **なぜ控えを残すか**：ChatWork送付分（1・3）は送りっぱなしだと後から「何を出したか」を追えない。控えを残すと提出証跡になる。
- **四半期の書き方**：`2026Q3`（2026年の第3四半期＝7〜9月）。領収書タスクは1・4・7・10月に発生。
- **タスク4は士業提出ではない**が、月次書類として同じ台帳・同じルールで管理すると迷いを減らせる。
- **クロコに保管を頼むとき**：この台帳の「保管先」と「控えファイル名」をそのまま渡せば、フォルダ作成（無ければ）→命名→移動まで指示できる。ただし「作った気」ではなく、移動後は必ず `ls` で実在を確認すること。
